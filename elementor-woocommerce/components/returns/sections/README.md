# Trocas e devolucoes em containers

Use estes blocos na ordem:

1. `01-hero.html` + `01-hero.css`
2. `02-policy-content.html` + `02-policy-content.css`

Esta pagina e institucional e nao depende de widget WooCommerce. Ela deve ser criada como pagina comum no WordPress, preferencialmente no slug:

```text
/politica-de-devolucao/
```

O texto em `02-policy-content.html` foi adaptado a partir da politica enviada pela MSN Suprimentos. Ele cobre direito de arrependimento, produto com defeito, reembolso e contato.

Antes de publicar, valide o texto final com o responsavel juridico/administrativo da loja. A base foi escrita para explicar o fluxo de atendimento, nao para substituir revisao legal.

Referencias oficiais para revisao:

- Codigo de Defesa do Consumidor, Lei 8.078/1990: https://www.planalto.gov.br/ccivil_03.old/leis/l8078compilado.htm
- Decreto do e-commerce, Decreto 7.962/2013: https://www.planalto.gov.br/ccivil_03/_ato2011-2014/2013/decreto/d7962.htm
- Senacon/MJSP sobre arrependimento em compras online: https://www.gov.br/mj/pt-br/assuntos/noticias/consumidor-tem-direito-ao-arrependimento-em-compras-on-line
