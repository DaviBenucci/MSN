# Loja, categorias e busca em containers

Use estes blocos na ordem:

1. `01-hero-search.html` + `01-hero-search.css`
2. `02-products-layout.html` + `02-products-layout.css` + `02-products-layout.js`

O bloco `02-products-layout` mantem filtros e grid como slots nativos. Insira o widget Woofilters/WBW no slot de filtros e Loop Grid Products, Products/Archive Products ou shortcode `[products]` no slot de produtos.
