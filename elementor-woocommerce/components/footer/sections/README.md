# Footer em containers

Crie um container pai com classe `msn-footer`.

Dentro dele, crie um container interno com classes:

```text
msn-container msn-footer__grid
```

Cole os blocos nesta ordem:

1. `01-brand.html`
2. `02-categories.html`
3. `03-service.html`
4. `04-policies.html`

Depois do grid, cole:

5. `05-bottom.html`

Use `00-footer-layout.css` para o layout geral e `00-footer-accordion.js` para as sanfonas mobile.

O bloco `04-policies.html` aponta para as paginas:

- `/politica-de-privacidade/`
- `/politica-de-devolucao/`
- `/finalizar-compra/`
- `/avalie-seu-pedido/`
